We have included a lot of useful utilities within AudioKit to help you get a project running quickly. None of these utilities are required by AudioKit for it to run (ie. none of them are imported into "AKFoundation.h") but you can pick and choose which features you need.

ClassDescription
AKSampler AKSampler is a simple multi-track capable recorder and playback system.
AKTools AKTools is our large suite of helper functions.

Instruments

Amplifiers
ClassDescription
AKAmplifier A monophonic amplification system with amplitude control.
AKStereoAmplifier A stereo amplification system with amplitude control.
Analyzers
ClassDescription
AKAudioAnalyzer An instrument that has two properties that detect the amplitude and frequency of an audio source signal.
Emulations
ClassDescription
AKBambooSticksInstrument An instrument that wraps the bamboo sticks physical model
AKMandolinInstrument An instrument that wraps the mandolin physical model
AKMarimbaInstrument An instrument that wraps the marimba physical model
AKPluckedStringInstrument An instrument that wraps the plucked string physical model
AKSleighbellsInstrument An instrument that wraps the sleighbells physical model
AKStickInstrument An instrument that wraps the stick physical model
StruckMetalBar An instrument that wraps the struck metal bar physical model
AKStruckMetalBarInstrument An instrument that wraps the tambourine physical model
AKVibraphoneInstrument An instrument that wraps the vibraphone physical model
Processors
ClassDescription
AKDelayPedal
AKPitchShifterPedal
AKReverbPedal A reverb processor and end point (outputs to the DAC)
Synthesizers
ClassDescription
AKFMOscillatorInstrument A synth that uses FM Synthesis to generate sounds, with frequency and amplitude defined
AKVCOscillatorInstrument An instrument for the Voltage-controlled oscillator operation

Plots

ClassDescription
AKAudioInputFFTPlot Plots the FFT of the audio input
AKAudioInputPlot Plots the incoming audio source signal, usually the microphone
AKAudioInputRollingWaveformPlot A Rolling Waveform for of the audio input
AKAudioOutputFFTPlot Plots the FFT of the audio output
AKAudioOutputPlot Plot the raw samples of the audio output to the DAC
AKAudioOutputRollingWaveformPlot A Rolling Waveform for of the audio output
AKFloatPlot Plots the floating point value given a minimum and maximum
AKInstrumentPropertyPlot Plot of the given instrument property
AKStereoOutputPlot Plot the raw samples of the audio output to the DAC as left and right signals
AKTablePlot Plots the values of the given table

User Interface Elements

ClassDescription
AKPropertyLabel Ties a label to the value of a property
AKPropertySlider A slider that sets the value of a property (within the property's bounds)
AKLevelMeter A level meter that can be drawn from any property
