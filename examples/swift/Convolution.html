--- title: AudioKit Examples - Swift Convolution header: Swift Convolution permalink: /examples/swift/Convolution/ layout: examples last-review-date: 2015/02/01 ---

This example uses a sampled audio file as the sound source, and applies a convolution effect to produce an interesting reverb and tone-shaped output. There are two impulse files that are used for the convultion that produce either a ringing bowl effect or a stairwell reverb. The user controls the mix of both of these effects with the original sound.

The first step is to load the loop and the impulse response files:

{% highlight ruby %} let file = String(NSBundle.mainBundle().pathForResource("808loop", ofType: "wav")!) let loop = AKFileInput(filename: file) let dish = String(NSBundle.mainBundle().pathForResource("dish", ofType: "wav")!) let well = String(NSBundle.mainBundle().pathForResource("Stairwell", ofType: "wav")!) {% endhighlight %}

Perform the convolutions:

{% highlight ruby %} let dishConv = AKConvolution(input: loop.leftOutput, impulseResponseFilename: dish) let wellConv = AKConvolution(input: loop.rightOutput, impulseResponseFilename: well) {% endhighlight %}

and mix the results:

{% highlight ruby %} let balance = AKMix(input1: dishConv, input2: wellConv, balance: dishWellBalance) let dryWet = AKMix(input1: loop.leftOutput, input2: balance, balance: dryWetBalance) {% endhighlight %}

Mac Version

The OS X and iOS versions of this example share the exact same AudioKit instrument files, but they use different UI elements to connect to the instrument's properties.
