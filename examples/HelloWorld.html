Every good programming framework needs a "Hello World" and this is AudioKit's version of Hello World! Basically, it is just the simplest and shortest amount of code that create an instrument and allow the user to start and stop that instrument from a button on the screen. The app simply plays an oscillator at 440Hz, but don't worry there are plenty of more complicated examples to try out once you have this one working.

This starter example is included in AudioKit in the Examples directory with one version for iOS and one for OSX. When you open the project you will see that there are two primary files that drive this example:

The ViewController.m file contains all of the AudioKit code as well and the Main.storyboard contains the button that activates methods in the ViewController.m file to control the instrument. There are main storyboard tutorials available online, so we'll just focus on the ViewController.m's AudioKit code.

Each important step is labeled with a comment and there only five steps! The first step is to make the AudioKit code available to this file which is accomplished by importing "AKFoundation.h":

{% highlight objective-c %} // STEP 1 : Import AudioKit's suite of classes #import "AKFoundation.h" {% endhighlight %}

Next, we're going to make an instrument available to every method in this class, so we create an instance of an AKInstrument which we will just call "instrument" in the implementation's instance variable declaration block:

{% highlight objective-c %} // STEP 2 : Set up an instance variable for the instrument AKInstrument *instrument; {% endhighlight %}

Inside the "viewDidLoad" method is a good place in the lifecycle of this app to create the internals of the instrument. Here we first allocate the memory and then create and connect an AKOscillator to that instrument. At this point, the instrument contains an oscillator, but nothing is connect to the audio output, so we also need to create an AKAudioOutput operation that takes the oscillator's output and sends it to your speakers.

{% highlight objective-c %} // STEP 3 : Define the instrument as a simple oscillator instrument = [[AKInstrument alloc] init]; AKOscillator *oscillator = [AKOscillator oscillator]; [instrument connect:oscillator]; AKAudioOutput *output = [[AKAudioOutput alloc] initWithAudioSource:oscillator]; [instrument connect:output]; {% endhighlight %}

Now that your instrument is created, it can be added to the orchestra. This lets AudioKit know that the instrument is fully defined and ready to be played. Since is the only instrument in our orchestra, we start the orchestra right after we add the instrument.

{% highlight objective-c %} // STEP 4 : Add the instrument to the orchestra and start the orchestra [AKOrchestra addInstrument:instrument]; [AKOrchestra start]; {% endhighlight %}

If you had other instruments, you would create and them here before starting the orchestra.

Finally, we just to need to connet the button from the Main.storyboard file to an action. The only AudioKit code here is [instrument play] and [instrument stop]. The other parts are just to update the UI to reflect the state of the instrument.

{% highlight objective-c %} // STEP 5 : React to a button press on the Storyboard UI by // playing or stopping the instrument and updating the button text. - (IBAction)toggleSound:(NSButton *)sender { if (![sender.title isEqual: @"Stop"]) { [instrument play]; sender.title = @"Stop"; } else { [instrument stop]; sender.title = @"Play Sine Wave at 440Hz"; } } {% endhighlight %}

And that's it!
