In this set of tutorials you will build the app you see demonstrated on the iPhone on this page. It will take you from setting up Audio Kit, building an instrument, and tying the instrument's properties to user interface elements. Since Apple has launched Swift we've created tutorials for creating iOS apps using Audio Kit in both Objective-C and Swift.
