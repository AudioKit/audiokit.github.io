GitHub Repository

There are two branches of AudioKit, the Master branch, and the Develop branch. The Develop branch is the default branch as it is where new features of AudioKit are added as they are finished... the very latest and greatest. If you're not comfortable living on the edge, then the Master branch is the most 'stable' option, as it is only changed when a new version of AudioKit is released.

  • AudioKitArchive Master branch - (24MB zip)
  • AudioKitArchive Develop branch - Development Branch (24.1MB zip)

Installing from command line (defaults to "master" branch):

{% highlight bash %} $ git clone https://github.com/audiokit/AudioKitArchive.git {% endhighlight %}

Installation Instructions

Installation of AudioKitArchive into your project differs depending on whether you're making an iOS or OSX project, whether or not it will be written in Swift, and what your choice of static vs. dynamic library is. There are two installation instruction files included with AudioKit:

  • INSTALL-iOS.md
  • INSTALL-OSX.md

Installing Xcode Documentation

Code completion, pop-ups, and sidebar information are automatically created in Xcode, but if you would like Apple-style API reference documentation, download the docset:

Un-tar it and place it in the Shared Documentation folder:

{% highlight bash %} tar -xf ~/Downloads/io.audiokit.AudioKit.docset.tar.gz -C ~/Library/Developer/Shared/Documentation/DocSets/ {% endhighlight %}
