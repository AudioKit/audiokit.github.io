AudioKit has two items in the resources folder, but each is full of important stuff:

AudioKit.plist Settings File

  • Audio Output (Default: dac)
  • Audio Input (Default: adc)
  • Sample Rate (Default: 44100)
  • Samples Per Control Period (Default: 64)
  • Number Of Channels (Default: 2)
  • Zero dB Full Scale Value (Default: 1)
  • Enable Logging By Default (false)
  • Prefix Csound Messages (false)
  • Enable Audio Input By Default (false)
  • Playback While Muted (false)
  • Perform To Disk (false)
  • MIDI Enabled (true)

AKSoundFile.bundle

This bundle contains a variety of audio samples:
  • Audio for use in AudioKit Tests and examples (808loop.wav, PianoBassDrumLoop.wav)
  • General MIDI Soundfont (GeneralMidi.sf2) and test soundfont (AcousticGuitar.sf2)
  • Some impulse response files (dish.wav, Stairwell.wav, shortpianohits.aif)
  • Default strike/pluck sounds (mandpluk.aif, marmstk1.wav)
  • Head-related transfer function data files (hrtf-44100-left.dat, hrt-44100-right.dat)

To use either, you must first drag from the AudioKit project to your project, making a copy if you so desire.
