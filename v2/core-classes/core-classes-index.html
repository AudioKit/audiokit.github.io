Foundation

The header file "AKFoundation.h" is what you have to import to use AudioKit. It includes all of AudioKit's core classes os well as all operations, instruments, and note classes. In addition, it includes all of the utlity classes of instruments, plots, and user interface elements.

Manager

This class handles all of the lower-level interaction with the audio engine. It rarely needs to be referred to expect for advanced users.

Orchestra

The orchestra is the object that you'll add your instruments to, makes sense doesn't it? You can also tell the orchestra to start or stop or when you've updated an instrument it needs to reload.

MIDI

AudioKit's MIDI functionality is stored here but could eventually be moved into "Utilities" since we certainly don't all need to use MIDI.

Classes

ClassDescription
AKFoundation This header file contains all the includes you might need for any AudioKit file
AKManager The manager controls all the interaction with the audio engine.
AKOrchestra This is the set of instruments available to play.
AKMidi The object that handles midi in and out information.
