AKSegmentArray.m

Overview

Trace a series of linear or exponential segments between specified points.

Instance Methods

addValue:afterDuration:

Adds a linear segment
{% highlight objective-c %}- (void)addValue:(AKConstant *)value afterDuration:(AKConstant *)duration{% endhighlight %}
  • value: Value after time given by duration.
  • duration: Duration in seconds.

addValue:afterDuration:concavity:

Adds another segment.
{% highlight objective-c %}- (void)addValue:(AKConstant *)value afterDuration:(AKConstant *)duration concavity:(AKConstant *)concavity{% endhighlight %}
  • value: Value after time given by duration.
  • duration: Duration in seconds.
  • concavity: A value greater than zero is an initially slower change and less than zero is a quicker intial change. Use zero for a linear change.

initWithInitialValue:targetValue:afterDuration:

Creates the exponential segment array and populates it with the minimum information. Use addValue:afterDuration to add more segments to the array. An optional release segment can be added with addReleaseToFinalValue:afterDuration.
{% highlight objective-c %}- (instancetype)initWithInitialValue:(AKConstant *)initialValue targetValue:(AKConstant *)targetValue afterDuration:(AKConstant *)duration{% endhighlight %}
  • initialValue: Starting value.
  • targetValue: Value after time given by duration.
  • duration: Duration in seconds of first segment.
