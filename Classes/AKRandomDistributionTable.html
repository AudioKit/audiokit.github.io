initBetaTypeWithSize:level:alpha:beta:

Create a random distribution table of the Beta type.

- (instancetype)initBetaTypeWithSize:(int)size level:(float)level alpha:(float)alpha beta:(float)beta

Parameters

size

Size of the table.

level

Level is the maximum amplitude of the signal varying from 0 to level or -level to level depending on the type.

alpha

If alpha is smaller than one, smaller values favor values near 0.

beta

If beta is smaller than one, smaller values favor values near level.
