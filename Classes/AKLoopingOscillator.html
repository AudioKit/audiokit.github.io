AKLoopingOscillator Class Reference

Inherits from AKAudio : AKParameter : NSObject
Declared in AKLoopingOscillator.h
AKLoopingOscillator.m

Overview

Read sampled sound (mono or stereo) from a table, with optional sustain and release looping, using cubic interpolation.

Warning: Not fully implemented yet: Currently no optional parameters are implemented.
