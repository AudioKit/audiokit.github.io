Inherits from AKStereoAudio : AKParameter : NSObject
Declared in AKLoopingStereoOscillator.h
AKLoopingStereoOscillator.m

Overview

Read sampled stereo sound from a table, with optional sustain and release looping, using cubic interpolation.

Warning: Not fully implemented yet: Currently no optional parameters are implemented.
