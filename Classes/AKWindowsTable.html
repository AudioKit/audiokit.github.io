Overview

Generates functions of different windows. These windows are usually used for spectrum analysis or for grain envelopes.

Window Types supported are:

  1. Hamming
  2. Hanning
  3. Bartlett (triangle)
  4. Blackman (3-term)
  5. Blackman - Harris (4-term)
  6. Gaussian
  7. Kaiser
  8. Rectangle
  9. Sync

initGaussianTypeWithBroadness:maximum:size:

Creates a Gaussian Windown Function Table

- (instancetype)initGaussianTypeWithBroadness:(float)windowBroadness maximum:(float)maximum size:(int)tableSize

Parameters

windowBroadness

Specifies how broad the window is, as the standard deviation of the curve; in this example the s.d. is 2. The default value is 1.

maximum

Absolute value at window peak point.

tableSize

Number of points in the table. Must be a power of 2 or power-of-2 plus 1.
