AKMIDI

public class AKMIDI

MIDI input and output handler

You add midi listeners like this: var midiIn = AKMidi() midi.openMIDIIn() midi.addListener(someClass) …where someClass conforms to the AKMIDIListener protocol

You then implement the methods you need from AKMIDIListener and use the data how you need.
