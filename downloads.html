Audio Kit Toolkit including example projects

We include a variety of sample projects alongside the toolkit so that you can get started right away. Here's how to get the complete package:

Installing from command line (defaults to "develop" branch):

> git clone git@github.com:audiokit/AudioKit.git

Archives are available:
  • Master Branch (19.5MB zip) - most stable
  • Development Branch (19.5MB zip) - latest and greatest

Audio Kit DocSet

Code completion, pop-ups, and sidebar information are automatically created in Xcode, but if you would like Apple-style API reference documentation, install the docset:

com.hearforyourself.AudioKit.docset.zip
(To install, unzip and place the resulting docset in your ~/Library/Developer/Shared/Documentation/DocSets/ folder.)
