Dynamics Processor

The AKDynamicsProcessoris both a compressor and an expander based on

Apple's Dynamics Processor audio unit. threshold and headRoom (similar to

'ratio' you might be more familiar with) are specific to the compressor,

expansionRatio and expansionThreshold control the expander.

{% highlight ruby %}import XCPlayground import AudioKit let file = try AKAudioFile(readFileName: "mixloop.wav", baseDir: .Resources) let player = try AKAudioPlayer(file: file) player.looping = true var effect = AKDynamicsProcessor(player) {% endhighlight %}

Set the parameters here

{% highlight ruby %}effect.threshold effect.headRoom effect.expansionRatio effect.expansionThreshold effect.attackTime effect.releaseTime effect.masterGain AudioKit.output = effect AudioKit.start() player.play() {% endhighlight %}
