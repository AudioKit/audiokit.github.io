Tracking Frequency

Tracking frequency is just as easy as tracking amplitude, and even

includes amplitude, but it is more CPU intensive, so if you just need amplitude,

use the amplitude tracker.

{% highlight ruby %}import XCPlayground import AudioKit {% endhighlight %}

Let's set up the volume to be changing in the shape of a sine wave

{% highlight ruby %}let volume = AKOperation.sineWave(frequency:0.2).scale(minimum: 0.5, maximum: 1) {% endhighlight %}

And let's make the frequency also be a sineWave

{% highlight ruby %}let minimum = Double(100) let maximum = Double(2200) let frequency = AKOperation.sineWave(frequency: 0.1).scale(minimum: minimum, maximum: maximum) {% endhighlight %}

So our oscillator will move around randomly in frequency and have a smoothly varying amplitude

{% highlight ruby %}let oscillator = AKOperation.sineWave(frequency: frequency, amplitude: volume) {% endhighlight %}

Connect up the the nodes

{% highlight ruby %}let oscillatorNode = AKOperationGenerator(operation: oscillator) let tracker = AKFrequencyTracker(oscillatorNode) let booster = AKBooster(tracker, gain: 1) let secondaryOscillator = AKOscillator() {% endhighlight %}

The frequency tracker passes its input to the output, so we can insert into the signal chain at the bottom

{% highlight ruby %}AudioKit.output = AKMixer(booster, secondaryOscillator) AudioKit.start() oscillatorNode.start() secondaryOscillator.start() {% endhighlight %}

And here's where we monitor the results of tracking the amplitude.

{% highlight ruby %}AKPlaygroundLoop(every: 0.1) { let amp = tracker.amplitude let freq = tracker.frequency secondaryOscillator.frequency = freq } {% endhighlight %}

This keeps the playground running so that audio can play for a long time
