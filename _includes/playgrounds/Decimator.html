Decimator

Decimation is a type of digital distortion like bit crushing, but instead of directly stating what bit depth and sample rate you want, it is done through setting "decimation" and "rounding" parameters.

{% highlight ruby %}import XCPlayground import AudioKit let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("drumloop", ofType: "wav") var player = AKAudioPlayer(file!) player.looping = true {% endhighlight %}

Next, we'll connect the audio sources to a decimator

{% highlight ruby %}var decimator = AKDecimator(player) {% endhighlight %}

Set the parameters of the decimator here

{% highlight ruby %}decimator.decimation = 0.5 // Normalized Value 0 - 1 decimator.rounding = 0.5 // Normalized Value 0 - 1 decimator.mix = 0.5 // Normalized Value 0 - 1 AudioKit.output = decimator AudioKit.start() player.play() {% endhighlight %}
