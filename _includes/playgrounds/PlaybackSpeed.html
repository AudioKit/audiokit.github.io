Playback Speed

Here we'll use the AKVariSpeed node to change the playback speed of a file (which also affects the pitch)

{% highlight ruby %}import XCPlayground import AudioKit let file = try AKAudioFile(readFileName: "mixloop.wav", baseDir: .Resources) {% endhighlight %}

Here we set up a player to the loop the file's playback

{% highlight ruby %}var player = try AKAudioPlayer(file: file) player.looping = true var variSpeed = AKVariSpeed(player) {% endhighlight %}

Set the parameters here

{% highlight ruby %}variSpeed.rate = 2.0 AudioKit.output = variSpeed AudioKit.start() player.play() {% endhighlight %}
