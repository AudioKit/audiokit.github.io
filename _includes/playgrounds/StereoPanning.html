Stereo Panning

Panning is a basic operation that is essential to mixing and

direction perception and it couldn't be easier with AKPanner.

{% highlight ruby %}import XCPlayground import AudioKit {% endhighlight %}

Set up the audio player

{% highlight ruby %}let file = try AKAudioFile(readFileName: "drumloop.wav", baseDir: .Resources) let player = try AKAudioPlayer(file: file) player.looping = true {% endhighlight %}

Route the audio player through the panner

{% highlight ruby %}var panner = AKPanner(player) AudioKit.output = panner AudioKit.start() player.play() {% endhighlight %}

Adjust the pan to smoothly cycle left and right over time

{% highlight ruby %} var time = 0.0 let timeStep = 0.05 AKPlaygroundLoop(every: timeStep) { panner.pan panner.pan = sin(time) time = time + timeStep } XCPlaygroundPage.currentPage.needsIndefiniteExecution = true {% endhighlight %}
