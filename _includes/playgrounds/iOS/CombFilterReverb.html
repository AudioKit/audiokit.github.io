Comb Filter Reverb

{% highlight ruby %}import XCPlayground import AudioKit let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("drumloop", ofType: "wav") var player = AKAudioPlayer(file!) player.looping = true var filter = AKCombFilterReverb(player, loopDuration: 0.1) {% endhighlight %}

Set the parameters of the Comb Filter here

{% highlight ruby %}filter.reverbDuration = 1 AudioKit.output = filter AudioKit.start() player.play() {% endhighlight %}

User Interface Set up
