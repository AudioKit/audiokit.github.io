Sean Costello Reverb

This is a great sounding reverb that we just love.

{% highlight ruby %}import XCPlayground import AudioKit let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("drumloop", ofType: "wav") var player = AKAudioPlayer(file!) player.looping = true var reverb = AKCostelloReverb(player) {% endhighlight %}

Set the parameters of the reverb here

{% highlight ruby %}reverb.cutoffFrequency = 9900 // Hz reverb.feedback = 0.92 AudioKit.output = reverb AudioKit.start() player.play() {% endhighlight %}
