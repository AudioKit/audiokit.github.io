Tracking Microphone Input

{% highlight ruby %}import AudioKitPlaygrounds import AudioKit let mic = AKMicrophone() {% endhighlight %}

Set the microphone device if you need to

{% highlight ruby %}if let inputs = AudioKit.availableInputs { try AudioKit.setInputDevice(inputs[0]) try mic.setDevice(inputs[0]) } let tracker = AKFrequencyTracker(mic, hopSize: 200, peakCount: 2_000) let silence = AKBooster(tracker, gain: 0) {% endhighlight %}

The frequency tracker passes its input to the output, so we can insert into the signal chain at the bottom

{% highlight ruby %}AudioKit.output = silence AudioKit.start() {% endhighlight %}
