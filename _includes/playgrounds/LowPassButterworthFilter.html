Low Pass Butterworth Filter

A low-pass filter takes an audio signal as an input, and cuts out the high-frequency components of the audio signal, allowing for the lower frequency components to "pass through" the filter.

{% highlight ruby %}import XCPlayground import AudioKit let bundle = NSBundle.mainBundle() let file = bundle.pathForResource("mixloop", ofType: "wav") var player = AKAudioPlayer(file!) player.looping = true var filter = AKLowPassButterworthFilter(player) {% endhighlight %}

Set the parameters here

{% highlight ruby %}filter.cutoffFrequency = 500 // Hz AudioKit.output = filter AudioKit.start() player.play() {% endhighlight %}
